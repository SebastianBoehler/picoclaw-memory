# Code of Conduct

## Our Standard

We want this project to be constructive, direct, and respectful.

Examples of behavior that contribute to a good environment:

- giving useful technical feedback
- staying specific and factual in disagreements
- respecting different experience levels
- focusing on the problem rather than the person

Examples of unacceptable behavior:

- harassment or personal attacks
- abusive or demeaning language
- bad-faith arguments
- doxxing or sharing private information
- spam or deliberately disruptive behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that do not align with this code of conduct.

## Scope

This code of conduct applies in project spaces such as issues, discussions, pull requests, and other public communication channels related to this repository.

## Reporting

For conduct concerns, contact the maintainer privately instead of escalating the situation in public.
